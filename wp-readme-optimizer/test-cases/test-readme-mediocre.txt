A simple contact form plugin for WordPress.

== Description ==

Contact Form Plus is a great plugin that allows you to add contact forms to your WordPress website easily. It's very powerful and flexible.

Features:
- Add contact forms to any page
- Email notifications
- Spam protection
- Customizable fields
- Works with all themes

Contact Form Plus is the best solution for your website contact needs. Download it today!

== Installation ==

Upload the plugin files to your WordPress plugins directory and activate it.

== Frequently Asked Questions ==

= How do I use it? =

Just install and activate the plugin, then use the shortcode [contact_form_plus] on any page.
